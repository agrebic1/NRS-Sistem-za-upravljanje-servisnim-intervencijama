# Sprint Goal

### **Sprint broj: 4** 

### **Sprint cilj:**  
Uspostaviti projektne i tehničke temelje sistema kroz definisanje Definition of Done, planiranje početnih release cjelina, izradu osnovnog skeletona projekta i organizaciju repozitorija, kako bi tim spremno prešao iz human-first u AI-enabled fazu razvoja.

### **Ključne stavke koje tim želi završiti:**  
- Definition of Done  
- Initial Release Plan  
- osnovni skeleton projekta  
- inicijalna struktura repozitorija  
- osnovni tehnički setup  
- završna validacija human-first faze  

### **Rizici i zavisnosti:**  
- nejasno definisani kriteriji završetka rada  
- neusklađenost oko prioriteta release cjelina  
- kašnjenje u tehničkom setupu i organizaciji repozitorija  
- nedovoljno stabilni temelji za prelazak u AI-enabled fazu
